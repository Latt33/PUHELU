# PUHELU — Project Description

PUHELU turns the SMS appointment confirmation that Finnish primary care patients already receive into a COPD screening opportunity. Over 300,000 Finns are estimated to have COPD, yet only 50,000 are actively treated. Patients complete a short adaptive questionnaire and vocal biomarker recording on their phone; when the assessment exceeds the threshold, a flag appears in the physician's existing patient information system through standard FHIR R4 integration. No new hardware, no software installations, no physician training, and no additional clinical time. The same integration pattern can be replicated across all four major Finnish EHR platforms, and per-patient screening cost is effectively zero, making it viable at any scale without dedicated resourcing.
